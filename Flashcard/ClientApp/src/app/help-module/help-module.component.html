Help Panel Visible because this is a demo; this would be removed on a prod site.
The create/edit deck page allows the user to create a flashcard deck.
The view page allows the user to view a deck, read notes by other users, and write a note (if they are logged in). A WYSIWYG is not provided here so I can encode any HTML characters.
The login page allows the user to log into their account. It also allows the user to navigate to the signup page.
The signup page allows the user to create an account. Password rules are:
  • Password must be at least 6 characters in length
  • Password must contain uppercase and lowercase letters, digits, and special characters
The home page shows all available flashcard decks on the right and a category filter on the left. If you resize the page, the number of columns of flashcard decks changes. Beyond a certain point, the flashcard decks will drop below the category filter dropdown.
